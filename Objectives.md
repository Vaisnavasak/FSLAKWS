**PROJECT OBJECTIVES**

1. To build a system that learns a keyword from 
   just 5 audio samples given by the user.

2. To make the system work for any language like 
   Tamil, Hindi, English, Telugu etc.

3. To search and find every occurrence of that 
   keyword inside a long audio file.

4. To display every timestamp where the keyword 
   was spoken in the audio.

5. To build a complete web application with login, 
   dashboard and keyword search features.

6. To store all search history and results 
   in a database.
