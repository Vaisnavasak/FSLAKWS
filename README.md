# FSLAKWS - Few Shot Language Agnostic Key Word Spotting System (FSLAKWS) for audio
This project is an audio keyword search system. It is employed to search and find a specific keyword in an audio. When we wanna find a specific word from a long audio, we have to do the following steps,
                        1. Give the word to be found with n (for example 5) audio samples
                        2. Present the audio file in which the keyword has to be searched
Finally, the system finds the word and present it with every timestamps it was spoken in the audio
It works for ANY word in ANY language.
# Simple understanding
It is like Ctrl + F which is used to find a specific word in a text. But here it search and find for a specific word in an audio
# Technology stack:
1. Frontend  : React.js
2. Backend   : Spring Boot (Java)
3. AI Engine : Python (Deep Learning)
4. Database  : MySQL
